# About LearnHub

LearnHub is a free, community-driven tutorial blog built on GitHub Pages using the Jekyll static site generator and the [Architect theme](https://pages-themes.github.io/architect/).

---

## Mission

Make high-quality, structured programming education accessible to everyone — no paywalls, no ads, no fluff.

Every lesson on LearnHub is:

- **Practical** — built around real code you can run immediately
- **Focused** — one concept per lesson, minimal detours
- **Progressive** — tracks build on each other in a logical order
- **Open** — all source on GitHub, contributions welcome

---

## Who is this for?

| Level                       | What you'll find                                         |
| --------------------------- | -------------------------------------------------------- |
| **Complete beginners**      | HTML, CSS, Git, and JavaScript tracks starting from zero |
| **Developers levelling up** | Intermediate JavaScript, REST APIs, and tooling guides   |
| **Anyone curious**          | Standalone tutorials and deep-dive articles on the blog  |

## Contributing

Found a mistake? Have a tutorial idea? Contributions are welcome.

1. Fork the repository on GitHub
2. Create a branch: `git switch -c feature/my-tutorial`
3. Write your lesson in `_posts/YYYY-MM-DD-title.md`
4. Open a Pull Request
