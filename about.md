# About LearnHub

LearnHub is a free, structured learning platform focused on **Artificial Intelligence, Agentic AI, and modern AI engineering**. Built on GitHub Pages, it is designed to take you from foundational AI concepts all the way to building and deploying autonomous AI agents in production.

---

## Mission

Make high-quality AI and agent engineering education accessible to everyone — no paywalls, no ads, no fluff.

Every lesson on LearnHub is:

- **Practical** — hands-on labs and real projects you build and run yourself
- **Focused** — one concept or framework per module, no unnecessary detours
- **Progressive** — phases build on each other from concepts to deployment
- **Open** — all source on GitHub, contributions welcome

---

## What is Agentic AI?

Agentic AI refers to AI systems that can **plan, reason, use tools, and take actions autonomously** to achieve a goal — rather than just answering a single question.

Key concepts covered on this site:

| Concept                 | What it means                                                            |
| ----------------------- | ------------------------------------------------------------------------ |
| **AI Agents**           | Autonomous systems that perceive, decide, and act in loops               |
| **ReAct pattern**       | Reason + Act — agents interleave thinking and tool calls                 |
| **Multi-agent systems** | Multiple specialized agents collaborating on complex tasks               |
| **Agentic RAG**         | Retrieval-Augmented Generation driven by an agent, not a static pipeline |
| **Tool use**            | Agents calling APIs, running code, searching the web, reading files      |
| **Memory & State**      | Short-term (in-context) and long-term (vector store) memory for agents   |
| **Observability**       | Tracing, logging, and evaluating agent runs in production                |

## What You Will Learn

LearnHub's AI curriculum is organized into four progressive phases:

**Phase 1 — Foundations**
Understand what makes a system "agentic", the core design patterns (Reflection, Tool Use, Planning, ReAct, ReWOO), and the ethical considerations around autonomous AI.

**Phase 2 — Core Frameworks**
Get hands-on with the most important agent frameworks: LangChain & LCEL for building chains, LangGraph for stateful graph-based agents, and LlamaIndex + Cohere for agentic RAG pipelines.

**Phase 3 — Advanced Agents**
Build multi-agent systems using CrewAI, LangGraph multi-agent, and Autogen. Understand when to use each, how to design agent roles, and how to orchestrate complex workflows.

**Phase 4 — Ops, No-Code & Cloud**
Make agents production-ready with observability (Langfuse, Langsmith), explore no/low-code tools (Langflow, Relevance AI, Wordware), and deploy agentic systems across AWS Bedrock, Azure OpenAI, and GCP Vertex AI.
